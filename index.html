1- ¿Cuáles son las principales funciones de un kernel en la arquitectura de un sistema operativo y cómo interactúa con el hardware?

El kernel es el núcleo del sistema operativo y se encarga de gestionar los recursos del sistema como la CPU, la memoria y los dispositivos de entrada/salida. Sus funciones principales incluyen la administración de procesos, la gestión de memoria, el control de dispositivos y la comunicación entre hardware y software. El kernel actúa como intermediario entre el hardware y las aplicaciones, permitiendo que el software funcione independientemente de los detalles del hardware.

2- ¿Cómo se gestionan los procesos y cuál es la importancia de la multitarea en la eficiencia del sistema?

La gestión de procesos implica crear, planificar, coordinar y terminar procesos dentro del sistema operativo. Esto se realiza mediante algoritmos de planificación que asignan tiempo de CPU a cada proceso. La multitarea permite que varios procesos se ejecuten aparentemente al mismo tiempo, mejorando el uso de los recursos del sistema y ofreciendo una experiencia más fluida al usuario. Es fundamental para maximizar la eficiencia y productividad del sistema.

3- ¿Qué medidas de seguridad se pueden implementar a nivel de sistema operativo para proteger una red, especialmente en entornos empresariales?

Entre las medidas de seguridad que se pueden implementar a nivel de sistema operativo están el uso de firewalls, sistemas de detección de intrusos (IDS), autenticación fuerte, políticas de control de acceso, cifrado de datos, actualizaciones regulares del sistema y el uso de software antivirus. En entornos empresariales, también se recomienda el uso de redes privadas virtuales (VPN), segmentación de red y monitoreo continuo para detectar y responder a amenazas en tiempo real.
